GWAAS: Leif GW As A Service

Do you need more Leif GW in your application but don't have access to Leif GW? Use GWAAS for all your Leif GW needs.


API Documentation

GET/api/v1/grunt: returns a random Leif GW grunt.
GET/api/v1/aaa: returns a random aaa, similar to the grunt but one can hear the letter A in it.
